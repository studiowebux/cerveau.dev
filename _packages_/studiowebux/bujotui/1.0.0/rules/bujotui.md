# Bujotui MCP — Bullet Journal

A `bujotui-mcp` server is available. It exposes a personal bullet journal with daily entries, collections, habits, and a future log.

**Use it when** the user wants to log daily tasks/events/notes, track habits, manage themed lists (reading, ideas, goals), or plan future months. Check the MCP tool schemas for parameters.

**Do not use it for** project task management (use mdplanner), code operations, or anything that belongs in the codebase.

Dates are `YYYY-MM-DD`. Entry indices are zero-based. Entries support state transitions: done, migrated, scheduled, cancelled.
